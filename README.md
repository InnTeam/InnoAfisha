<h1 align="center">InnoAfisha</h1>

## Project setup

```
npm install
npm run dev
```

## Description

<p align="center">
  InnTeam team took on the task of creating
such a convenient web service. There, everyone will be able to view all the
information about upcoming events, and add their favourite events to
“favourites”. It will be possible to publish events only with the approval of the
site administration, which will consider the events from the point of view of the
correctness of the presentation of information, the security and legality of the
event, etc.
</p>

# Features of our Project

1. Login and Logout
2. Separate page with all discription of the event
3. Ability to add the event into favorites
4. Ability to view all events
5. Ability to filter event by type
6. Ability to view favorites events

# Frameworks or technology

We use the Svelte with TypeScript for our project. Also we write some backend on Jango for authorization and saving events
